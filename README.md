# ShanhaiContinent Reforge

A reworked version of the original **ShanhaiContinent** mod, focused on fixing major issues in the source code and improving gameplay experience.

## Fixes
This version resolves several critical bugs from the original mod:

- Fixed **Soul Ring rendering errors**
- Fixed **Soul Ring stacking issues**
- Fixed **Soul Ring dismantling problems**
- Fixed several other minor bugs

## Improvements
- Added and modified various gameplay functions
- Increased the mod's **unique experience gain by 50x**
- General code improvements and stability fixes

## New Features
- Added **Vietnamese language support**
- Introduced a new Martial Soul: **Black Flame Asura**
